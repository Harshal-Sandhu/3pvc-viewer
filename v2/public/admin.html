Add compliance record

Writes one record to the selected site's compliance measurement using InfluxDB line protocol. Empty fields are skipped. api_version is required.

Add site

A site = one InfluxDB instance you want to view. Provide where it lives (IP + port), which database and measurements to read from, and a short name to identify it in the viewer dropdown.

Delete site

Delete ? This removes the entry from sites.json. InfluxDB data is not touched.
